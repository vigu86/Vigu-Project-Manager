This is the Timesheets section.

In the Timesheets section, users create and manage their weekly work timesheets.

Active week

The grey section below indicates which week is now active by showing the date of that week's Sunday.

Below the grey area, the user can see the entries created for the given day. The active day is indicated by the highlighted "day of week" button.

Adding new week

The user can add a new week by clicking the "Add Week" button.

Changing active week

The user can change the active week by clicking the "Change Week" button.

Adding new timesheets

If the timesheet of a given day is not yes created, the user needs to click the "Add Timesheet" button and provide the start time of their work.

Adding new entries

When the timesheet is created, the user can then add new entries by clicking the "Add Entry" button. In the modal which shows up, the user provides the time when they finished the task, as well as the project's name and phase.

Updating and deleting entries

The user can update or delete existing entries by using the action buttons on the right-hand side of the table.

Summary

The Summary tab shows all unique entries created in the active together with summarized hours spent on them.
